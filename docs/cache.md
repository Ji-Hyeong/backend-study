# Cache Lab

## 학습 질문

- cache-aside는 DB 갱신과 캐시 삭제 사이에서 어떤 불일치가 생기는가?
- cache stampede는 왜 발생하고 어떻게 완화하는가?
- TTL은 정합성 문제를 해결하는가, 지연시키는가?
- 로컬 캐시와 Redis 캐시는 장애와 스케일아웃 관점에서 무엇이 다른가?

## 실험 계획

- 느린 원본 조회를 만들고 캐시 적용 전후의 호출 수를 비교한다.
- 동시 요청에서 같은 key가 한 번만 계산되도록 제어하는 실험을 추가한다.
- stale cache를 의도적으로 만들고 갱신 전략을 비교한다.
